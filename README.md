# Kitty Terminal Installer

A simple bash script to install and configure the Kitty terminal emulator.

## Compatibility
This script is designed for **Linux** environments. It is specifically intended for **Debian-based distributions**, such as:
* Debian
* Ubuntu
* Kali Linux
* Linux Mint

## Usage
1. Clone the repository or download the script.
2. Make it executable: `chmod +x install_kitty.sh`
3. Run the script: `./install_kitty.sh`
